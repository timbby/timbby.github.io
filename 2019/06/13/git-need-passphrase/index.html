WHAT

git操作远程分支(git pull,git push等)时提示enter passphrase for key '~/.ssh/id_rsa'

WHY

之前有同事操作git时，拉代码需要输入密码，然后用ide拉代码就没有问题，当时就没太在意。但是后来又遇到了同样的问题，这次是在服务器上配置的id_rsa，出现了同样的问题，每次都要输入密码，于是就查了一下解决办法。

HOW
